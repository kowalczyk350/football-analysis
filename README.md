# football-analysis
Analiza meczu Granada CF – FC Barcelona (La Liga 2020/2021, match_id=3773565). Wizualizacja podań Barcelony: zielone strzałki oznaczają udane podania, czerwone nieudane. Dane pochodzą ze StatsBomb Open Data, a boisko i grafika zostały przygotowane w Pythonie z użyciem mplsoccer.
